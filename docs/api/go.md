# slop-ai (Go)

```bash
go get github.com/devteapot/slop/packages/go/slop-ai
```

## Main types

- `Server` and `ScopedServer` for providers
- `Consumer` for subscribing to existing providers
- `Node`, `Props`, `ActionOpts`, `Handler`, and `HandlerFunc`
- WebSocket, Unix socket, and stdio transport helpers

## Best fit

- `net/http` services
- local daemons and desktop helpers
- CLI tools
- Go-based protocol inspectors and agents
